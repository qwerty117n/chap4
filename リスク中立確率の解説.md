# リスク中立確率はなぜ存在するの？～高校生にもわかる解説～

## まずイメージをつかもう

明日の天気が「晴れ・くもり・雨」の3パターンあるとします（これが「状態が3つ」ということ）。

明日の天気によって、あなたの持っている株の値段が変わります。今日S円の株が明日には、

- 晴れなら Sa円（一番上がる）
- くもりなら Sb円
- 雨なら Sc円（一番下がる、a > b > c なので一番安い）

になるとします。一方、銀行に預けたお金（債券）は、天気に関係なく必ず 1円が (1+r)円に増えます（r は金利、安全に増える分）。

ここで使える「道具」は株と債券の2つだけ。でも天気のパターン（状態）は3つあります。これが「資産の数2 ＜ 状態の数3」という状況です。

## リスク中立確率って何？

価格を決めるとき、「晴れになる確率」「くもりになる確率」「雨になる確率」のような、ちょっと特殊な確率（q_a, q_b, q_c）を考えます。これは本当の天気予報の確率ではなく、

> 「この確率を使って将来のお金の平均をとると、今の値段とちょうど合う」

という、いわば帳尻合わせ用の確率です。この確率を使うと、新しい金融商品の値段もきれいに計算できる、というのが本文の(5.8)式・(5.9)式の主張でした。

## なぜ困るのか（方程式が足りない）

この帳尻合わせ用の確率は、次の2つの式を満たさないといけません。

1. 確率は全部足すと1
   q_a + q_b + q_c = 1

2. 株の値段が合う
   q_a × a + q_b × b + q_c × c = 1 + r

式は **2本** しかないのに、知りたい未知数（q_a, q_b, q_c）は **3つ** あります。連立方程式は「式の数 ≧ 未知数の数」じゃないと、答えがピッタリ1つに決まりません。だから答えは1つに決まらず、条件を満たす (q_a, q_b, q_c) の組が **無数にある** ことになります。これが「資産の数 ＜ 状態の数」のときに起きる問題（市場の不完備性）です。

## でも「存在する」ためには条件がある

無数にあるとはいえ、q_a, q_b, q_c はどれも確率なので **0以上(マイナスはダメ)** でなければいけません。「マイナスにならない解が存在するか」を考えると、答えはこうなります。

> **c < 1 + r < a**

これが満たされていれば、マイナスにならない (q_a, q_b, q_c) の組が見つかります。

## なぜこの条件が必要なのか（直感）

これは「ノーフリーランチ（タダ飯はない）」の話です。もしこの条件が崩れたら、リスクゼロで確実に儲かる方法（さや取り、套利）が生まれてしまいます。

**ケース1：1+r が a より大きい場合**
銀行に預けるほうが、株が一番良い結果（晴れ）になった場合よりもさらに儲かることになります。つまりどんな天気でも銀行預金の方が株より得、ということです。そうなったら誰も株を買わず、株を売って全員銀行に預けるはずです。これはリスクなしで必ず儲かる「変な状況」で、現実の市場ではすぐに価格が調整されてこんな状態は続きません。

**ケース2：1+r が c より小さい場合**
どんな天気でも株の方が銀行より得なので、みんな銀行のお金を借りて株を買おうとします。これも「リスクなしで必ず儲かる」変な状況です。

だから、まともな（さや取りができない）市場であるためには、銀行の増え方(1+r)が、株の一番悪い結果(c)より大きく、一番良い結果(a)より小さい、というちょうど中間の位置にいる必要があります。これが **c < 1+r < a** です。

## まとめ

- 資産(株・債券)が2つしかないのに、天気(状態)が3パターンあると、帳尻合わせの確率は1つに決まらず、**無数の答え** が出てくる（市場が不完備）。
- ただし、その中にマイナスにならない確率の組が存在するためには **c < 1+r < a** が必要。
- この条件が崩れると、リスクゼロで儲かる「タダ飯」が生まれてしまい、市場としておかしな状態になる（套利機会の発生）。
- 新しい金融商品（オプションなど）の値段も、無数にある確率のどれを選ぶかで変わってしまうので、套利が無いという条件だけからは値段が1つに決まらず、**上限と下限の範囲** しか分からない。
