# 二項モデルにおけるリスク中立確率の存在条件

## モデルの設定

２時点1期間モデルで現在の株価を $p$、$t=1$ の株価を２つの状態 $P_{up}$、$P_{down}$ としたとき、状態確率（リスク中立確率）$q^*$ は、割引した期待株価が現在の株価に等しくなるように定義されます。

リスクフリーの粗収益率を $R = 1 + r$ とすると：

$$p = \frac{1}{R}\big[q^*\, P_{up} + (1-q^*)\, P_{down}\big]$$

これを $q^*$ について解くと：

$$q^* = \frac{Rp - P_{down}}{P_{up} - P_{down}}$$

---

## 存在条件

$q^*$ が確率として意味を持つ（$0 < q^* < 1$）ためには、$P_{up} > P_{down}$ のもとで次が必要十分です：

$$P_{down} < Rp < P_{up}$$

すなわち $(1+r)p$ が２つの状態株価の間にあること。アップ率 $u = P_{up}/p$、ダウン率 $d = P_{down}/p$ を使えば、より見やすく：

$$d < R < u \quad(\text{つまり } d < 1+r < u)$$

---

## これは無裁定条件そのもの

この条件は「裁定機会が存在しない（no-arbitrage）」ことと同値です。直感的には：

- $R \le d$ のとき（リスクフリー収益が最悪ケースの株式収益以下）  
  → 借入れて株を買えば確実に儲かる

- $R \ge u$ のとき（リスクフリー収益が最良ケースの株式収益以上）  
  → 株を空売りして安全資産に投資すれば確実に儲かる

どちらも裁定になるため、$d < R < u$ が成り立つときに限って $q^*$ が $(0,1)$ の中に**唯一**存在します。一期間二項モデルでは市場が完備なので、この $q^*$ は一意です。
