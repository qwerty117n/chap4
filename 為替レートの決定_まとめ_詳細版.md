# 3. 為替レートの決定

## 3.1 長期理論——購買力平価（PPP）説

### PPPの基本的な考え方
- 一物一価の法則を国際的に拡張した考え方
- 「同じ財・サービスはどの国でも同じ価格になるはず」という発想に基づく
- 財の移動が完全に自由で輸送コストがゼロなら、内外価格差は生じない

### 絶対的PPP
- 式：e = P / P*（e：為替レート、P：自国物価、P*：外国物価）
- 1単位の自国通貨で買える財の量＝1単位の外国通貨で買える財の量、となるように為替レートが決まる
- 物価は財の消費バスケット（複数財の加重平均）として表される

### 相対的PPP
- 式：S_t = π_t − π*_t（為替レートの変化率＝自国インフレ率−外国インフレ率）
- 絶対的PPPが成立すれば相対的PPPも成立するが、逆は必ずしも成り立たない
- 「為替レートの絶対水準」を説明するのは難しいが、「変化率」なら説明しやすい、というのがポイント
- インフレ率の高い国の通貨は、低い国の通貨に対して減価する

### 具体例：ビッグマック指数
- 英国のエコノミスト誌が公表している有名な実用例
- 各国のビッグマックの価格を比較することで、PPPに基づく為替レートを簡易的に算出
- 実際の為替レートとPPPベースの為替レートを比較することで、各国通貨の割高・割安を国際比較できる

## 3.2 実質為替リスク・名目為替リスク

### 名目為替レートと実質為替レート
- 名目為替レート：通常観察される市場の為替レートそのもの
- 実質為替レート：物価水準で調整した為替レート　X = S ×（P*/P）
- 実質為替レートは「2国間の財の相対的な価値」を表す

### なぜ区別が重要か
- 為替リスクを考えるとき、名目為替リスクと実質為替リスクは別物
- PPPが常に成立する世界なら、実質為替レートは一定→外国資産投資は実質的に為替リスクなし
- 現実の世界ではPPPは短期的には成立しない理由：
  - 貿易財に対する裁定取引が不十分（取引コスト・関税など）
  - 非貿易財（サービスなど）の存在
  - 国ごとの消費バスケットの違い

### 短期と長期での違い
- 短期：PPPからの乖離が大きい→名目為替レートの変動≒実質為替レートの変動（実質為替リスクが大きい）
- 長期：PPPが概ね成立する傾向→実質為替レートの変動は小さい（実質為替リスクは小さい）
- 実質為替レートが収束していく基準値を「為替アンカー（錨）」と呼ぶ

### 実務上のインプリケーション
- 短期の投資家（数ヶ月〜1年程度）は名目為替レートの変動リスクをそのまま負う
- 長期の投資家（年金基金など）は、実質為替リスクは限定的と考えられる
- → 投資のホライズン（期間）によって為替ヘッジの必要性が変わってくる

## 3.3 短期理論——カバーなしの金利平価

### 短期理論の発想
- 短期的な為替レートの変動は、財市場（PPP）ではなく資産市場の無裁定条件によって決まると考える
- 国際的な資産（債券など）への投資のリターンが、内外で均等化するように為替レートが決まる

### カバー付き金利平価（CIP）
- 式：1 + r = (F/S) × (1 + r*)
- F：フォワードレート、S：スポットレート、r：自国金利、r*：外国金利
- フォワード市場で為替リスクを完全にヘッジ（カバー）した場合、内外資産への投資リターンが等しくなるという条件
- CIPは現実の為替市場でほぼ成立している（裁定取引によって維持される）

### カバーなしの金利平価（UIP）
- 式：E(S_{t+1}) = S_t × (1+r)/(1+r*)
- フォワードでカバーせず、将来の期待為替レートを使って内外資産のリターンを比較したもの
- 投資家がリスク中立的であれば、フォワードレートは将来の期待為替レートの不偏推定値になる
- すなわち：F ≈ E(S_{t+1})

### CIPとUIPの違いのポイント
- CIP：常に成立する（無裁定条件、確実な関係）
- UIP：投資家の期待や為替リスクスケープレミアムに依存（理論的な仮定が必要）

---

## 3.4 カバーなしの金利平価の実証——フォワード・プレミアム・パズル

### 検証方法
- 回帰式：S_t = a + b・f_t
- UIPが成立するなら理論上 a = 0, b = 1 となるはず

### 実証結果
- 多くの実証研究で b は1よりかなり小さく、しばしばマイナスになる
- これは「フォワードレートが高い（金利が高い）国の通貨ほど、実際には増価する」という、理論と逆の現象を示す
- この現象は「フォワード・プレミアム・パズル」と呼ばれている

### パズルに対する2つの説明
1. **為替リスクスケープレミアムの存在**
   - 投資家は為替リスクを取ることに対して追加的なリターン（リスクプレミアム）を要求する
   - これがUIPからの乖離を生む
2. **市場の非効率性・投資家の過剰反応**
   - 投資家のリスクに対する反応が市場の歪みを生み、為替レートが過剰に変動する
   - 金融政策の変化への市場参加者の過剰反応なども一因とされる

### 重要な論文
- Froot–Thaler（1990）、福田・五十嵐（1997）などがこのパズルに関する代表的な研究

## 3.5 為替のリスクプレミアム

### 為替リスクスケープレミアム（SRP）の定義
- 式：SRP ≡ r* + E[(S1 − S0)/S0] − r
- 外国資産へ投資し、為替リスクをカバーせずに保有した場合の期待リターンと、自国の安全資産（フリーリスク資産）への投資リターンとの差
- 分散化によって消去できない、為替市場全体に内在するリスクに対する対価

### なぜリスクプレミアムが生じるのか
- 為替レートの変動は、株式や債券などの個別資産の危険資産に対するリスクと同様に、ポートフォリオ全体のリスクに影響を与える
- 投資家がどの国の通貨を「安全」と考えるかによって、リスクプレミアムの大きさ・符号が変わる

### 国によって非対称になる点（重要）
- 日本の投資家が米ドル資産に投資する場合のリスクスケープレミアムと、米国の投資家が円資産に投資する場合のリスクスケープレミアムは、一般に等しくならない
- これは、各国の投資家にとっての「自国通貨」が異なるため
- 為替リスクスケープレミアムを統一的に説明する決定的な理論は、現時点でも確立されていない（依然として研究上のパズル）

## まとめ（3章前半の流れ）
- 長期：PPP（購買力平価）によって為替レートの「変化率」を説明
- 為替リスクは「名目」と「実質」を分けて考える必要があり、投資ホライズンによって意味が異なる
- 短期：金利平価（CIP・UIP）によって資産市場から為替レートを説明
- 実証ではUIPは成立しにくく（フォワード・プレミアム・パズル）、その背景に為替リスクスケープレミアムの存在が指摘される
- 次節（4節）では、このリスクプレミアムを国際的な資産価格理論として体系化した「国際CAPM」を扱う
