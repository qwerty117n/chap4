# リスク中立確率の存在条件 ～一般化（n資産・m状態）～

## 1. 設定

教科書の記法に合わせる。状態は j = 1, 2, …, m の m 通り、資産は i = 1, 2, …, n の n 種類とし、

- 資産 i の t=0 での価格を p_i
- 資産 i が状態 j で生む t=1 でのキャッシュフローを D_{i,j}

とする。このとき(5.8)式は次のように書ける。

p_i = Σ_{j=1}^m q_j D_{i,j}　（i = 1, …, n）

行列・ベクトル表記では、

**p = D q**

- p：n×1 の価格ベクトル
- D：n×m の支払行列（D_{i,j} を並べたもの）
- q：m×1 の「状態価格ベクトル」（q_j は状態 j の成分の単価）

## 2. n < m のとき何が起きるか

これは「未知数 q ∈ R^m が m個あるのに、式（制約条件）は n本しかない」という連立1次方程式である。線形代数の一般論から、

- n本の式が独立であれば、解の集合は **m − n 次元のアフィン部分空間** になる（解が1つに決まらず、無数にある）。
- これが「資産の数 < 状態の数」のときに価格付けが一意に決まらない理由である。

## 3. リスク中立確率が「存在する」ための条件（一般形）

q_j 自体はそのまま確率になるわけではないが、無リスク資産（債券、状態によらず 1 → 1+r となる資産）が含まれる場合、リスク中立確率は

π_j = (1+r) q_j

として定義され、Σ π_j = 1 となるように q が選ばれる。このとき、

**π_j ≥ 0（すべての j について）**

を満たす q（同値に π）が存在することが、リスク中立確率の存在条件である。

### 資産価格付けの基本定理（一期間モデル）

次の2つは同値である。

1. p = Dq、 q ≥ 0（あるいは q > 0）を満たす状態価格ベクトル q が存在する。
2. 市場に**裁定機会（アービトラージ）が存在しない**。すなわち、ポートフォリオ θ = (θ_1, …, θ_n) で
   - θ'D ≥ 0（どの状態でも損をしない）かつ θ'D ≠ 0（少なくとも1つの状態で得をする）
   - θ'p ≤ 0（コストがゼロ以下）

   となるものが存在しない。

これは線形計画法における **ファルカスの補題（Farkas' lemma）** の応用である。「ある連立不等式に非負解が存在しない」ことと「もう一方の連立不等式に解が存在する」ことが同値になる、という双対性を使っている。直感的には、「タダ飯（リスクなしで儲かる組み合わせ）が作れない」ことと「整合的な状態価格（リスク中立確率）が存在する」ことが表裏一体だ、ということである。

## 4. 設問の場合への当てはめ（n=2, m=3）

設問は n=2（株式・債券）、m=3（状態 a, b, c）の特殊ケースである。

D = [[Sa, Sb, Sc], [1+r, 1+r, 1+r]]　、　p = [S, 1]

q = (q_a, q_b, q_c)、あるいは π = (π_a, π_b, π_c) = ((1+r)q_a, (1+r)q_b, (1+r)q_c) を考えると、制約は次の2本になる。

- Σ π_j = 1
- π_a・a + π_b・b + π_c・c = 1+r

この2本の式（n=2）に対して未知数は3つ（m=3）なので、解は1次元の自由度を持つ集合（直線）になる。その直線上で π ≥ 0 となる範囲が存在するための必要十分条件は、

**c < 1+r < a**

である。これは基本定理の「θ'D ≥ 0 かつ θ'p ≤ 0 となる θ が存在しない」という条件を具体的に書き下したものに他ならない。たとえば 1+r ≥ a なら、θ = (株を1単位売り、その代金で債券を買う) というポートフォリオがどの状態でも損をしない裁定になり、定理の対偶から非負の状態価格は存在しなくなる。

## 5. 一般の n < m での結論

- n < m のとき、no-arbitrage条件（基本定理の条件2）を満たせば、非負の状態価格ベクトル q（リスク中立確率 π）は **存在するが一意ではない**（m − n 次元の自由度を持つ集合になる）。
- 新しい金融商品の価格は、この自由度のどの q を選ぶかで変わってしまうため、no-arbitrage だけからは一意に決まらず、取りうる価格の **上限・下限（super-replication／sub-replication 価格帯）** としてしか特定できない。
- 一意の価格付けを行うには、
  (a) 独立な資産を m 個まで増やして市場を**完備化**する、
  (b) 効用関数や均衡条件など**追加の情報**を導入する、
  のいずれかが必要になる。
