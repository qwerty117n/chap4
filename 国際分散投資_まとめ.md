# 国際分散投資の基礎（2.2〜2.4）

## 国際分散投資の基礎
- 商品、REIT等の不動産、ヘッジファンドなどのオルタナティブ投資、中国・インド・ロシア等の新興国（エマージング市場）の重要性が増加
- これらは先進国の株式・債券との相関が低く、同じ期待リターンに対するリスクを減少させる効果がある
- メリットは投資対象資産のタイプを多様化するほど大きくなる

## 2.3 国際的な資産リターン間の相関係数の決定要因
- 分散投資の利益は国内・外国資産間の相関に依存（相関が低いほど効果大）
- 相関を決める2要因：①マクロ経済の相互依存度　②国際資本市場の統合度

### ① マクロ経済の相互依存度
- グローバル化により多国籍企業化、調達・販売のグローバル化が進展（例：自動車産業の現地生産シフト）
- 株価は国内要因だけでなく世界共通要因・同業種要因の影響を受ける
- 先進国間の協調的政策（例：2000年代の協調利下げ）も各国独自要因を希薄化させる

### ② 国際資本市場の統合度
- 完全統合＝取引障壁（規制・コスト・税金・情報）がゼロの状態
- 統合が進むと、一物一価・類似リスクへの類似プレミアムが成立（→国際CAPMへ）
- 現実には障壁が残るため、ニュースが部分的にしか反映されず各国特有の動きが生まれ、相関は低めに保たれる
- 近年は規制緩和・コスト低下で統合が進展中→相関上昇・分散効果縮小の方向

### 相関の時間変動
- 市場が大きく変動する時期（ブラックマンデー、2007年以降の金融危機など）には相関が一時的に上昇
- 分散投資メリットが大幅に低下するため、ボラティリティの時間変動への注意が必要

## 2.4 国際分散投資の新たな手法
- 伝統的手法＝国単位の分散（カントリー・アロケーション）には限界がある
- 経済統合が進んだ地域（EU圏・北米圏など）内での分散は相関が高く効果が小さい
- グローバル化により、意図しない業種・スタイルの偏りが発生しやすい
- 業種分散が不十分だと、国レベルで分散してもリスク低減効果が不十分になる
- 実証研究：カントリー要因の重要性低下／業種要因の重要性上昇
- → ファクター・モデル（カントリー以外に地域・業種・スタイル等を使用）の活用が有効＝グローバル・ファクター・モデル
- 債券にもグローバル利回り曲線の水準・傾き等をファクターとするモデルが使われている
